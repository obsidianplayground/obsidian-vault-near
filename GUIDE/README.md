# NEAR OBSIDIAN PLUGIN
a plugin for obsidian that adds near blockchain interaction for function calls and transactions

---

#### SET UP
- enable the core webviewer plugin
- sign into the wallet at https://wallet.intear.tech/

#### COMMANDS

TOGGLE NETWORK (mainnet/testnet)
- toggles the network
- saves network to local storage

SIGN IN
- sign in so you can do transactions
