# NEAR OBSIDIAN VAULT
an obsidian vault for testing the obsidian near plugin

---

Download the plugin
```sh
cd path/to/vault
mkdir .obsidian/plugins
cd .obsidian/plugins

git clone hhttps://github.com/obsidianplayground/obsidian-plugin-near
cd obsidian-sample-plugin
bun i
bun run build
```
