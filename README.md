# NEAR OBSIDIAN VAULT
an obsidian vault for testing the obsidian near plugin

---

#### DOWNLOAD THIS VAULT
```sh
git clone https://github.com/obsidianplayground/obsidian-vault-near
```


#### DOWNLOAD THE PLUGIN
```sh
cd path/to/vault
mkdir .obsidian/plugins
cd .obsidian/plugins

git clone https://github.com/obsidianplayground/obsidian-plugin-near
cd obsidian-sample-plugin
bun i
bun run build
```

---

#### ABOUT THIS PLUGIN
the plugin adds near transaction support, and can be used via the command palette.
to use it must set up a markdown file with transaction properties.

see [GUIDE](GUIDE/README.md)
